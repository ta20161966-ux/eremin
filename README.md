# Movie Library

**Автор:** [Имя Фамилия]

**Описание:**
Приложение для хранения и фильтрации информации о фильмах. Позволяет добавлять фильмы, фильтровать по жанру и году, сохранять и загружать данные из JSON.

**Примеры использования:**
1. Запустите приложение.
2. Введите данные о фильме и нажмите «Добавить фильм».
3. Используйте фильтры для поиска по жанру или году.
4. Данные сохраняются в файл movies.json при закрытии.
